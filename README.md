# An algorithm to compute the answer for a competitive programming contest

This is my quick solution for weird Finnish competitive "coding" contest, which can be found at:
https://github.com/wunderdogsw/wundernut-vol2

In order to keep things simple, the related Java code is compiled in console. 

(Please notice that the source text needed to compute the answer is not publicly available anymore. But you can use any other Finnish text source file for testing.)
